# Benjamin Graham: Skill Extraction Report

## Scoring Methodology

Each skill is evaluated on 5 criteria, scored 1-10:

| Criterion | Description |
|-----------|-------------|
| **Uniqueness** | How distinctively Graham can this skill be performed? |
| **Actionability** | Can this produce concrete, usable output? |
| **Universality** | Does this apply broadly across contexts? |
| **Teachability** | Can the framework be clearly communicated? |
| **Demand** | How often would users need this skill? |

**Maximum Score:** 50 points
- **HIGH Priority (40+):** Create standalone skill PROMPT.md
- **MEDIUM Priority (30-39):** Include in persona, no standalone file
- **LOW Priority (below 30):** Reference only
