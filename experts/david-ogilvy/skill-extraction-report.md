# David Ogilvy - Skill Extraction Report

## Scoring Criteria

Each skill is evaluated on 5 criteria (10 points each, 50 points max):

| Criterion | Description |
|-----------|-------------|
| **Actionable** | Can be applied immediately with clear steps |
| **Invocable** | Has a clear trigger phrase or use case |
| **Scoped** | Well-defined boundaries; not too broad or narrow |
| **Reusable** | Applicable across many situations and contexts |
| **Valuable** | Provides significant insight or advantage |

**Priority Levels:**
- HIGH: 40+ points
- MEDIUM: 30-39 points
- LOW: Below 30 points
