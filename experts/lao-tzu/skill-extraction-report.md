# Lao Tzu - Skill Extraction Report

## Scoring Criteria

Each skill is evaluated on 5 criteria, scored 1-10:

| Criterion | Description |
|-----------|-------------|
| **Actionability** | Can this skill be invoked with clear inputs and outputs? |
| **Distinctiveness** | Does this skill offer unique value not found elsewhere? |
| **Practical Value** | Does this solve real problems people face? |
| **Persona Alignment** | Does this skill authentically represent Lao Tzu's teachings? |
| **Repeatability** | Can this skill be applied consistently across different situations? |

**Priority Thresholds:**
- HIGH: 40+ points (create standalone skill)
- MEDIUM: 30-39 points (include in persona, no standalone)
- LOW: Below 30 points (mention only)
